# Battery-Finder-Program

## 概要
step1及びstep2において任意のカメラのブランドとモデルを選び、step3でアクセサリーの消費電力を入力することで、これらの条件に対応する電池の一覧と各電池の持続時間を表示します。

## URL
このURLからアプリケーションを見ることができます。</br>
https://naonao3.github.io/Battery-Finder-Program/</br>
